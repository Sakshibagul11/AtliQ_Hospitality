# AtliQ_Hospitality - Power BI
Atliq Hospitality is a premium hotel chain operating across multiple cities in India. This project focuses on analyzing hotel performance data to help management make data-driven decisions related to revenue, occupancy, and customer behavior.
The dashboard provides a clear view of key business metrics such as Revenue, Occupancy Rate, ADR (Average Daily Rate), RevPAR, and Booking Trends.
